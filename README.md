# Sonu Kumar — Portfolio

A premium, animated personal portfolio built with **Flutter Web** — showcasing my work as a Senior Flutter Developer & UI/UX Engineer.

🔗 **Live:** _coming soon_

## ✨ Features

- Clean **light theme** with soft shadows, gradient accents and a custom **SVG hero illustration**
- Smooth **scroll-reveal** animations, animated skill bars, rotating typewriter roles
- Sections: Hero · About · Skills · Experience timeline · Projects · Contact
- Fully **responsive** (desktop / tablet / mobile)
- Clickable project cards linking to live **Play Store / App Store** apps
- Frosted-glass sticky navigation

## 🛠 Tech

`Flutter` · `Dart` · `google_fonts` · `flutter_svg` · `url_launcher`

## 🚀 Run locally

```bash
flutter pub get
flutter run -d chrome
```

## 📦 Build for web

```bash
flutter build web --release
# output in build/web/
```

## 📁 Structure

```
lib/
├── data/         # all portfolio content (edit here)
├── theme/        # colors, gradients, fonts
├── sections/     # hero, about, skills, experience, projects, contact
├── widgets/      # reusable animation + layout widgets
└── utils/        # helpers
assets/svg/       # hero illustration
```
